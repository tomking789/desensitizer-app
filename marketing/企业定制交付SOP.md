## 3. 构建安装包

```bash
# 基础版（仅 Logo + 词库）
python scripts/build_enterprise.py \
    --customer-name "客户全称" \
    --customer-short-name "客户简称" \
    --logo "客户logo.png" \
    --terms "客户词库.csv"

# 有额外要求的定制版
python scripts/build_enterprise.py \
    --customer-name "客户全称" \
    --customer-short-name "客户简称" \
    --logo "客户logo.png" \
    --terms "客户词库.csv" \
    --product-name "自定义产品名" \
    --edition-name "专属版"
```

输出：
- `dist/企业资料脱敏工具-{客户简称}-定制安装包-{日期}.exe`
