## Important Compliance Notes

- Do not distribute the packaged executable without also distributing or
  clearly offering the corresponding source code.
- Keep `LICENSE`, `AGPL-3.0.txt`, `NOTICE.md`, and this file with the source.
- Keep `DISCLAIMER.md` and `INSTALLATION.md` with distributed packages.
- If the program is modified and distributed, distribute the modified source
  under the same AGPL terms.
- If the software is provided as a network service, make sure users can obtain
  the corresponding source code as required by the AGPL.
